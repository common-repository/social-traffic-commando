== Description ==

Syndicate you wordpress posts to facebook, twitter, plurk, tumbr, friendfeed, livejournal, Post your wordpress posts to facebook, twitter, plurk, tumbr, friendfeed, livejournal, diigo, delicious and linkedin.

Share and reshare the posts
Reblog the posts, tweet and retweet the posts


== Installation ==

1. Upload `social-traffic-commando` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add content to posted to the social site through the post add/edit screen
4. Add accounts and wait for reshares, reblogs, retweets
